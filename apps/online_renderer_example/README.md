## Quick Start

### 1. Clone and Setup

```bash
# Navigate to the simple_example directory
cd apps/online_renderer_example

# Get dependencies
flutter pub get
```

### 2. Run the Application

```bash
# Create the necessary runtime
flutter create .

# Run on connected device/emulator
flutter run

# Or run in debug mode
flutter run --debug
```
